# HR Analytics Dashboard

## Overview

This project is an interactive HR Analytics Dashboard built using Microsoft Power BI. It provides insights into workforce metrics such as employee attrition, demographics, department distribution, and salary analysis through interactive visualizations.

## Key Metrics

- Total Employees
- Active Employees
- Attrition Count
- Attrition Rate
- Average Age
- Average Experience

---

## Dashboard Features

- Employee Distribution by Department
- Attrition by Department
- Attrition by Salary Slab
- Attrition by Gender
- Age Group Distribution
- Department Employee Count
- Job Role and Satisfaction Matrix
- Interactive Age Group Slicer
